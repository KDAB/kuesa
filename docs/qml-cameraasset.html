CameraAsset QML Type

Convenience wrapper to retrieve an asset from the Kuesa::SceneEntity::cameraCollection. More...

  • List of all members, including inherited members

Detailed Description

 import Kuesa 1.3 as Kuesa

 Kuesa.CameraAsset {
     name: "cameraAssetName"
 }

It is technically equivalent to:

 Kuesa.Asset {
     collection: sceneEntity.cameras
     name: "cameraAssetName"
 }
