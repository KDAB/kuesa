    GLTF2MaterialEffect QML Type

    GLTF2MaterialEffect is a subclass of Qt3DRender::QEffect and is the base class of all GLTF2 based material effect subclasses. More...

    Since: Kuesa 1.2
    Instantiates: GLTF2MaterialEffect
    • List of all members, including inherited members

    Properties

    • alphaCutOffEnabled : bool
    • doubleSided : bool
    • opaque : bool
    • useSkinning : bool

    Detailed Description

    It holds properties to specify which features are enabled such as:

    • Vertex Skinning
    • Opaque
    • Alpha Cut off
    • Double Sided

    Property Documentation

    alphaCutOffEnabled : bool

    Returns whether alphaCutOffIsEnabled or not.

    This property was introduced in Kuesa 1.2.


    doubleSided : bool

    Returns whether the meshes to be rendered with this effect are double sided.

    This property was introduced in Kuesa 1.2.


    opaque : bool

    Returns whether the content to be draw is opaque or transparent.

    This property was introduced in Kuesa 1.2.


    useSkinning : bool

    Returns whether vertex skinning is enabled or not.

    This property was introduced in Kuesa 1.2.
