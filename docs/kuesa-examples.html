Kuesa Examples

QML Examples

Kuesa Car-Scene QML Example

Demonstrates the use of the Kuesa API to import a glTF2 file and extract assets from it.

Kuesa Iro-Materials-Gallery QML Example

Demonstrates the appearance of the different Kuesa Iro Materials that can be bundled in a glTF file.

Kuesa Music Box QML Example

Demonstrates the use of the Kuesa API to import a glTF2 file and play animations.

Kuesa ToneMapping QML Example

Demonstrates the use of the Kuesa API to import a glTF2 file and control its appearance by using different tone mapping algorithms.

Kuesa Many Ducks C++ Example

Demonstrates the use of the Kuesa C++ API to import a glTF2 file and extract assets from it.
